# fake.bio

*A modern link-in-bio and portfolio platform, serving as a powerful alternative to guns.lol*

---

## Overview

fake.bio is a comprehensive profile creation platform that enables users to build stunning, feature-rich link-in-bio pages and portfolios without writing code. With over 970,000 users on platforms like guns.lol, Radium aims to provide a more customizable and modern alternative for creating your online presence.

## Key Features

### **Profile Customization**
Create fully customizable profiles with backgrounds, avatars, custom cursors, and audio players. Express your unique style with extensive personalization options.

### **Link Management**
Add unlimited links to your social media, websites, projects, and more. Organize them with categories and custom styling for better navigation.

### **Template System**
Choose from professionally designed templates or create your own. Share templates with the community and browse user-created designs.

### **File Hosting**
Fast and secure file hosting for images, documents, and media. Direct linking support with reliable CDN infrastructure.

### **Analytics Dashboard**
Track profile views, link clicks, visitor statistics, and engagement metrics. Monitor your growth with comprehensive analytics.

### **Discord Integration**
Connect your Discord account to display status, show rich presence, and enable advanced social features directly on your profile.

## Getting Started

### Prerequisites

- Node.js 16.0 or higher
- npm or yarn package manager
- Git for version control

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/radium.git

# Navigate to the project directory
cd radium

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`

---

## Usage Guide

1. **Create Account** - Sign up with email or social accounts
2. **Choose Template** - Browse template gallery or start from scratch
3. **Customize Profile** - Set background, avatar, colors, and fonts
4. **Add Links** - Insert your social media and important links
5. **Enhance Features** - Add audio, custom cursor, and Discord integration
6. **Publish & Share** - Go live and share your profile link

### Technology Stack

- **Frontend**: React 18 with TypeScript for type safety
- **Styling**: Tailwind CSS for utility-first design
- **Backend**: Supabase for database and authentication
- **Build Tool**: Vite for fast development and building
- **Animations**: GSAP for smooth transitions and effects
- **State Management**: React Context for global state

## Why Radium?

While guns.lol has established itself as a popular platform with over 970,000 users, Radium offers several advantages:

- **Open Source**: Full transparency and community contributions
- **Self-Hosted**: Complete control over your data and platform
- **Customizable**: Extensive theming and plugin system
- **Privacy Focused**: No data mining or selling user information
- **Modern Tech Stack**: Built with the latest web technologies
